MRiceTower Neon Drive - working export

IMPORTANT:
1. Download BOTH zip files.
2. Unzip BOTH into the SAME folder.
3. Keep index.html and the audio folder next to each other.
4. Open index.html. If your browser blocks local audio, run this in that folder:
   python3 -m http.server 8000
   then open http://localhost:8000

This version removes the missing /assets dependency and uses local audio with a hosted fallback.
